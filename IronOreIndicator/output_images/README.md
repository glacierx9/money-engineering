# Output Images Directory

This directory contains all visualization outputs from the IronOreIndicator project.

## Structure

Each run of the visualization script creates a timestamped subfolder:
```
output_images/
├── 2025-10-31_run/                    # Manual run from Oct 31
├── 2025-11-01_143022_run/             # Auto-timestamped run
└── 2025-11-01_150315_run/             # Another auto-timestamped run
```

## Files Generated per Run

Each run typically generates:

1. **multi_instrument_nm_grid.png** - N×M grid showing all instruments and indicators
2. **smoothed_dashboard_{market}_{code}.png** - Detailed dashboard for each instrument with:
   - Smoothed price trends (HMA, KAMA, TEMA, SuperSmoother)
   - Smoothed RSI and MACD
   - Signal strength visualization
   - Buy/sell signal markers
   - Signal interpretation table

3. **instrument_dashboard_{market}_{code}.png** - Individual instrument analysis (optional)

## Example Output

For a run with 6 instruments (DCE: i, m | SHFE: rb, cu | CZCE: SR, TA):
```
2025-11-01_143022_run/
├── multi_instrument_nm_grid.png
├── smoothed_dashboard_DCE_i<00>.png
├── smoothed_dashboard_DCE_m<00>.png
├── smoothed_dashboard_SHFE_rb<00>.png
├── smoothed_dashboard_SHFE_cu<00>.png
├── smoothed_dashboard_CZCE_SR<00>.png
└── smoothed_dashboard_CZCE_TA<00>.png
```

## Naming Convention

- **Timestamp Format**: `YYYY-MM-DD_HHMMSS_run` (e.g., `2025-11-01_143022_run`)
- **Market Codes**: DCE (lowercase), SHFE (lowercase), CZCE (UPPERCASE)
- **Instrument Codes**: Logical contracts ending in `<00>` (e.g., `i<00>`, `rb<00>`, `SR<00>`)
